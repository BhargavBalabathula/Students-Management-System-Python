# Student Management System (Python + Tkinter + MySQL)

## Features
- Add Student
- Update Student
- Delete Student
- View Students
- Logging System

## Tech Stack
- Python
- Tkinter
- MySQL

## How to Run
1. Install dependencies
2. Setup MySQL database
3. Run `main.py`
